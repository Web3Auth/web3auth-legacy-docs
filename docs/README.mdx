# Web3Auth Legacy Documentation

Welcome to the Web3Auth Legacy Documentation portal. This site contains documentation for
specialized Web3Auth SDKs that are maintained for specific use cases but are not intended for
general usage.

:::note

If you've been directed here by your Web3Auth representative, please consult with them before
proceeding with any integration to ensure you're using the appropriate solution for your needs.

:::

:::caution

If you're looking for our main documentation and generally recommended SDKs, please visit
[https://web3auth.io/docs](https://web3auth.io/docs).

:::

## Specialized SDKs for Advanced Use Cases

The SDKs documented here may:

- Address specific edge cases
- Support legacy implementations
- Require additional configuration
- Have limited support compared to our main SDKs
